# ТребоВАНИя

### Основное
1) Надо написать тест, состоящий из нескольких вопросов
2) У каждого вопроса ровно два варианта ответа, один из которых правильный
3) Чтобы ответить на вопрос, надо навести персонажа на кнопку с ответом
4) Персонаж управляется стрелочками

### Вопросы
1) Первый вопрос должен быть "куда ты прыгнешь", вместо вариантов ответа две ямы, а на фоне игуана (сидит в траве)
2) Второй вопрос должен быть "Какую кнопку нажмешь?", вместо вариантов две кнопки
3) У первого вопроса правильный ответ слева, у второго - справа
4) Если человек ответил правильно на оба вопроса, в конце он съедает чебурек. Если на оба неправильно, происходит взрыв. Иначе тест просто завершается
5) Если человек очень долго отвечает на вопрос, его съедает динозавр и тест провален

### Оформление
1) Человечек должен быть с ногами и руками
2) Размер человечка примерно как размер кнопки - 200 * 200 пикселей
3) У человечка должна быть большая голова
4) Лицо человечка берем с упаковки йогурта Фруате
5) Текст вопроса оформляется большим шрифтом, чтобы он занимал примерно 50% экрана по ширине
